Gryn Wasabi ERP System

For our final capstone project, my groupmates and I collaborated with a real client—Gryn Wasabi Sushi Bar—to develop a comprehensive ERP (Enterprise Resource Planning) system tailored to their business operations. The system includes 7 integrated modules: Order Tracking, User Access Control, Order Management, Finance Management, Branch Management, CRM Module & HR Management, and Inventory Management. We built the system using modern web technologies such as Firebase, Node.js (NPM), TailwindCSS, HTML, CSS, and JavaScript. This project gave us hands-on experience in full-stack development, client engagement, and building scalable, real-world business solutions.
